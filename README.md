# PZ_4

## Ответы на вопросы

1.  Html создает фундамет сайта, а css создает его силь.
2.  GitHub Pages - бесплатный хостинг сайтов, который превращает файлы из репозитория в веб-сайт.
3.  Список используемых тего/атрибутов:
      1. Head - заголовок и техическая информация
      2. body - содержимое страницы
      3. Footer - Имя автора сайта/раздела, его контакты
      4. Header - загаловок раздела
      5. Nav - контейнер для ссылок
      6. Ul - маркированный сисок
      7. Li - Элеманты списка
      8. Main - основное содержимое докумена
      9. H1 - заголовок 1 уровня
      10. Img - подгрузка изображения
      11. Title - сам заголовок
      12. Link - подключение ссылки.
      13. P - текстовый абзац
      14. Style - определяет стиль определенного обьекта на странице.
      15. border-bottom - определят стиль границы
      16. Padding - задает расстояние от внутреннего края рамки до содержимого
      17. Margin - регулирует пространство от границы текущего элемента до внутренней границы его родительского элемента
      18. border-bottom - стиль границ
      19. border-radius - радиус скругления границы
      20. float - определяет выравнивание элемнта
      21. box-shadow - добавляет тень элемента
      22. overflow - управляет отображением содержания блочного элемента при его выходе за границы размеров
      23. H2 - заголовок 2 уровня
      24. color - цвет элемента
      25. content -  позволяет вставлять генерируемое содержание в текст веб-страницы
      26. display - определяет, как элемент должен быть показан в документе
      27. clear - Устанавливает, с какой стороны элемента запрещено его обтекание другими элементами
      28. div - выделяет элемент документа и позволяет изменять его содержимое
